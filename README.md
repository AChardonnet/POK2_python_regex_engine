# POK 2 : un moteur regex en python

## Installation

1. Create a virtual environement:

```bash
python -m venv venv
```

2. Activate it:

* Windows (PowerShell): `.\venv\Scripts\activate.ps1`
* Linux/macOS: `source venv/bin/activate`

3. Install dependancies:

```bash
pip install -r requirements.txt
pip install -e .
```
